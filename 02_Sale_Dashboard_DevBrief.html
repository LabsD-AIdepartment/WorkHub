Dev BriefAuth & Access Control
v1.0
AUTH Authentication Rules
Login: Username + Password → bcrypt verify → return JWT (expire 8h)
ทุก API request ต้องมี Authorization: Bearer {token} header
Admin สามารถ CRUD users + assign/change role ทุก role รวมถึง Admin
Refresh token endpoint สำหรับ extend session
เตรียม OAuth2 flow สำหรับ SSO ในอนาคต (อย่า hardcode auth logic)
