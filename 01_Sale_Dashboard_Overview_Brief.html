What it does
ระบบทำอะไร

ดึงข้อมูลจาก Facebook Ads, Shopee, Lazada, TikTok Shop และระบบ Order ของบริษัทมาเก็บไว้ใน Database กลาง แล้วแสดงผลผ่าน Web Dashboard พร้อมคำนวณตัวชี้วัดทางการตลาดและการเงินโดยอัตโนมัติ

Brief Summary
สรุป Brief ทั้งหมด
Overview (หน้าหลัก)
  • ยอดขายรวม GMV ทุก Platform — Filter แยก Facebook / Marketplace / ดูสัดส่วนได้
  • ต้นทุนค่าโฆษณารวม / ต้นทุนต่อการซื้อ
  • ROAS / ROI / AOV
  • Date Range Filter — ไม่เกิน 1 เดือนย้อนหลัง
Ads Meta — 22 บัญชี (รองรับถึง 25)
  • Total Spending ของทุกบัญชี / ทุกสินค้า
  • Spending แยกบัญชีโฆษณา / แยกสินค้า
  • Click (ทักข้อความ) รวมทุกบัญชี / ทุกสินค้า
  • Click แยกบัญชีโฆษณา / แยกสินค้า
  • Conversion (การซื้อ) รวมทุกบัญชี / ทุกสินค้า
  • Conversion แยกบัญชีโฆษณา / แยกสินค้า
  • CVR — %ปิดการขาย (Click : Conversion)
  • Metric ต่อเนื่อง: ROAS / CPR (ต้นทุนต่อคำสั่งซื้อ) / CPA (ต้นทุนต่อการทัก)
Marketplace — Shopee 6 / Lazada 5 / TikTok 3 ร้าน (รวม 14)
  • ยอดขายรวม GMV ทุกร้านค้า — Filter รวม/แยกแต่ละ Platform / แยกรายร้านย่อยได้
  • ต้นทุนค่าโฆษณา / คำสั่งซื้อ / ต้นทุนต่อการซื้อ / AOV
  • ROAS / ROI
  • จำนวน Click
  • CVR — จำนวนการซื้อต่อจำนวน Click
  • Platform Fee — ดึงจาก API ถ้าได้ / fallback เฉลี่ย 25% ของ GMV
Team Sale Admin
  • CVR รวมทุกทีม (Click : Conversion)
  • CVR แยกทีม
  • CVR แยกสินค้า
  • ยอดขายรวมทั้งหมด / ยอดขายแยกทีม
Product
  • รายการ Orders ทั้งหมด — Filter Facebook / Marketplace ได้
  • ยอดขายแยกรายสินค้า — Filter Facebook / Marketplace ได้
  • จำนวนโปรโมชัน Top List แยกรายสินค้า
  • ต้นทุนสินค้า (COGS) — ดึงจาก Google Sheet, Auto Sync รายวัน + กด Sync ทันทีได้
System Requirements
  • ดึง API อัตโนมัติรายวัน — Real-time ยิ่งดี
  • กำหนดสิทธิ์การเข้าดูข้อมูลของแต่ละ User ได้
  • Login ด้วย Username / Password — สร้างและจัดการ User ในระบบนี้เลย
  • Admin จัดการสิทธิ์ได้ทุก Role รวมถึง Admin เองได้
  • ออกแบบให้รองรับการเชื่อมต่อกับ ERP Project ที่กำลังพัฒนา
  • Internal Order System — เชื่อมต่อ DB โดยตรง (รายละเอียด TBC)
